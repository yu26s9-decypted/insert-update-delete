# Insert, Update, Delete

SQL exercises practicing DML (Data Manipulation Language) operations — the statements that modify data in the **Northwind** database.

## About This Project

Created at **Year Up United — Learn to Code Academy (LTCA)** as part of the SQL curriculum. While earlier exercises focused on reading data, this project covers how to write, change, and remove records in a relational database.

## What I Learned

- Adding new rows to a table with `INSERT INTO`
- Modifying existing records with `UPDATE` and `SET`
- Removing records with `DELETE`
- The importance of `WHERE` clauses when updating/deleting (to avoid changing all rows)
- How foreign key relationships affect what you can insert or delete
- Working with the Northwind database schema to insert realistic test data

## Tech Stack

- **SQL** (MySQL)
- **Northwind** sample database

## Topics Covered

| Statement | Description |
|-----------|-------------|
| `INSERT INTO` | Add new rows to a table |
| `UPDATE ... SET` | Modify existing column values |
| `DELETE FROM` | Remove rows from a table |
| `WHERE` | Target specific rows during mutation operations |
